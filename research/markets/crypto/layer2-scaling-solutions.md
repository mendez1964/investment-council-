# Layer 2 Scaling Solutions — Complete Guide

## Why Layer 2 Exists
Ethereum processes ~15 transactions per second on L1. Visa processes 24,000. At scale, Ethereum L1 fees become prohibitive ($10-100+ per transaction in peak periods). Layer 2 solves this without changing Ethereum's security or decentralization.

## How Layer 2 Works
L2s process transactions off Ethereum mainnet but settle proofs or data ON Ethereum L1:
- Users transact on L2 (fast, cheap)
- L2 batches thousands of transactions
- Posts compressed proof or data to Ethereum L1 (expensive, but shared cost)
- Result: same security, fraction of cost

## Two Main L2 Technologies

### Optimistic Rollups (Arbitrum, Optimism, Base)
- Assume all transactions are valid unless challenged
- 7-day challenge period for withdrawals to L1
- Cheaper to build, more EVM-compatible
- Currently largest TVL among L2s

**Arbitrum:** Largest L2 by TVL. Strong DeFi ecosystem (GMX, Aave, Uniswap).
**Optimism:** OP Stack powering Base, Zora, and many others. Strong governance.
**Base (Coinbase):** Fastest growing. Built on OP Stack. Retail-friendly.

### ZK-Rollups (zkSync, Starknet, Polygon zkEVM)
- Generate cryptographic proof that all transactions are valid
- Instant withdrawal to L1 (no challenge period)
- More complex to build, harder to achieve full EVM compatibility
- Vitalik's preferred long-term solution

**zkSync Era:** Largest ZK-rollup by TVL. EVM compatible.
**Starknet:** Cairo language, not EVM. Most technically advanced.
**Polygon zkEVM:** Polygon's ZK solution, EVM-equivalent.
**Linea (Consensys):** Enterprise-focused ZK rollup.

## How to Analyze an L2
Key questions:
1. **TVL trend:** Growing or declining?
2. **Transaction volume:** Actual usage or wash trading?
3. **Unique active wallets:** Organic user growth?
4. **Revenue to Ethereum L1:** Protocol revenue metric
5. **dApp ecosystem:** Quality of applications deployed
6. **Bridge security:** Largest historical hack vector
7. **Centralization:** Who controls the sequencer?

## Bridge Risk
Most L2 hacks happen at bridges (moving assets between chains):
- 2022: $600M+ in bridge hacks (Ronin, Wormhole)
- Risk: bridge holds assets in smart contract → if contract exploited, funds lost
- Safer: native bridges (official Arbitrum/Optimism bridges)
- Riskier: third-party bridges (Multichain, etc. — some failed)

## L2 Tokens as Investments
**ARB (Arbitrum), OP (Optimism):**
- Governance tokens for their respective L2 networks
- Not direct claims on protocol revenue (currently)
- Valued on speculation that governance rights = future value accrual
- Risk: Ethereum itself could build native L2 features making them obsolete

**Key metric for L2 token thesis:** Revenue generated by L2 (fees collected), currently mostly going to L1 settlement costs not token holders.
